객체

* 객체 선언 및 사용

- 객체는 중괄호{}를 사용하여 생성
- 중괄호 안에 객체 필요한 속성들 정의
- '속성명 : 속성값'의 형태로 정의 (key:value)
- 속성값으로는 모든 자료형의 값을 담을 수 있음 

        변수 = {
            속성명 : 속성값,
            속성명 : 속성값,
            속성명 : 속성값,
            ....
        }

* 속성에 접근시 특이케이스(속성명을 호출할 대 반드시 문자열로만 하는 경우)


* 객체에서의 반복문

객체가 가지고 있는 모든 속성에 순차적으로 접근하고자 한다면 반복문 사용가능
 단, 일반 for문으로는 불가. for in으로 사용 가능
